• Introduction

    • In this application, you can generate a random string which contains numbers, letters, and a set of special characters. 

    • Besides generating the random string, you can also perform several filtering functionalities such as removing a certain types of characters within the generated string.

  • Description

    • This is a WorkFlow Console Application

  • Testing instruction

    • Steps to run the application

      • Start the console application
      • Enter a numerical length which will be used to generate the indicate length of the random string
      • If you enter 0, which will prompt you to re-enter the length
      • Then you will have 2 choices, one is to remove all the characters except the numbers, then sort the numbers in the random string in descending order; The another choice is to remove all the all characters except the letters, then sort the letters in the random string in alphabetical order.
      • Once you finish either of choice above, you can press enter to exit the application
